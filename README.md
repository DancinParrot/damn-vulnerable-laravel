## Damn Vulnerable Laravel

Damn Vulnerable Laravel (DVL) is an intentionally vulnerable web application that is developed based on the Laravel framework. The DVL project is still under development, and only includes the following vulnerablities:

- File Upload Vulnerability with Race Condition

Hence, one may practice web application pen-testing using DVL.

## Deployment

1. Create a `.env` file from the provided `.env.bkup` file.

2. Then, Damn Vulnerable Laravel may be deployed using docker:
    ```bash
    docker build -t acme/app .
    docker run --rm -it -e VT_KEY=<VirusTotal API Key> -p 80:80 -p 8085:8085 acme/app
    ```
